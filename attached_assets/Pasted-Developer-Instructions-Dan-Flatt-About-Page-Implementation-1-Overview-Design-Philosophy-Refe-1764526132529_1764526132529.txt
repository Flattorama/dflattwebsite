Developer Instructions: Dan Flatt - About Page Implementation

1. Overview & Design Philosophy

Reference Site: Cyd Stumpel About Page
Core Aesthetic: Maximalist typography, high-contrast, playful but executive, fluid scroll-triggered animations.

Key Technical Requirements:

Framework: React / Next.js (Preferred) or plain HTML/GSAP.

Animation Library: GSAP (GreenSock) with ScrollTrigger is highly recommended for the complex header scaling and card fanning effects.

Responsive: All animations must degrade gracefully on mobile (stack vertically instead of fan out if screen width < 768px).

Scroll: Parallax scroll speed (bubbles move faster/slower than the text).

3. Global Scroll Interaction: "The Shrinking Headers"

This is the signature interaction of the reference site.

Logic

As the user scrolls down to a new section (e.g., "About Me", "Services"), the large section title must transition from Part of the content flow to Fixed/Sticky Header.

Initial State: The Section Title (e.g., "ABOUT ME") appears large (approx 8vw font size) in the center of the viewport as you arrive at the section.

Trigger: As the user scrolls past the title.

Action:

The title becomes position: fixed or sticky.

It scales down significantly (from 8vw to ~1.5vw).

It moves to the Top Center of the screen.

Exit: When the user reaches the end of that section, the title scrolls away (un-sticks) or dissolves, replaced by the next section's title which repeats the process.

6. Section 3: "Ventures" (The Card Stack)

Reference: The "Websites" section on Cyd's site.

Sticky Header Text: "VENTURES"

The Card Stack Animation (CRITICAL)

Initial State: A stack of 3-4 cards (Case Studies) are piled directly on top of each other in the center/right of the screen.

Scroll Action: As the user scrolls down the section, the cards fan out.

Card 1: Rotates -5deg, translates Left.

Card 2: Stays Center.
